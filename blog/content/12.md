## 配置

**邮箱**<br>
1. 全局安装<br>
`git config --global user.email 'your_email@xxx.com'`<br>
2. 在每一个你需要设置的仓库执行<br>
`git config user.email 'your_email@xxx.com'`<br>
例如：在你的某个目录下，执行：<br>
```bash
cd ~/project
for repo in $(ls); do cd $repo; git config user.email 'your_email@xxx.com'; cd ..; done
```

**用户名**<br>
用户名的设置也和邮箱相同
