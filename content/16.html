# typeof & instanceof 的区别

TIP

简单区分一下 typeof 和 instanceOf。

typeof 是一元运算符，用来返回操作数类型的字符串。语法——

const name = true;
console.log(typeof name === "boolean"); // true
类型 结果
Undefined "undefined"
Null "object" (历史遗留问题)
Boolean "boolean"
Number "number"
String "string"
Symbol (ECMAScript 6 新增) "symbol"
宿主对象(由 JS 环境提供) Implementation-dependent
函数对象 ( [[Call]] 在 ECMA-262 条款中实现了) "function"
任何其他对象 "object"

instanceof 运算符用于测试构造函数的 prototype 属性是否出现在对象的原型链中的任何位置。语法——

function Student(name, age) {
  this.name = name;
  this.age = age;
}
const student = new Student("Bob", 11);
console.log(student instanceof Student); // true

由于 typeof 只能判断类型，而且数组和对象返回的都是 object，这时候就需要 instanceof 看来判断——

const arr = [];
console.log(arr instanceof Array); // true

封装一个专门进行类型判断的函数：

function getDataType(obj) {
  if (obj === null) {
    return "null";
  } else if (typeof obj === "object") {
    if (obj instanceof Array) {
      return "array";
    } else {
      return "object";
    }
  } else {
    return typeof obj;
  }
}
