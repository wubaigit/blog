# new 和 prototype

TIP

简单聊一下 new

new 运算符创建一个用户定义的对线类型的实例，或者具有构造函数的内置对象的实例

function Student(name, age) {
  this.name = name;
  this.age = age;
}
const student = new Student("Bob", 11);

创建一个用户自定义的对象需要两步——

  • 通过编写函数来定义对象类型
  • 通过 new 来创建对象实例

创建一个对象类型，需要创建一个指定其名称和属性的函数；对象的属性可以指向其他对象

当代码 new Foo() 执行时——

  1. 一个继承自 Foo.prototype 的新对象被创建
  2. 使用指定的参数调用构造函数 Foo，并将 this 绑定到新创建的对象。new Foo 等同于 new Foo()，
  3. 由构造函数返回的对象就是 new 表达式的结果，如果构造函数没有显式返回一个对象，则使用步骤 1 创建的对象。
function Foo() {
  this.name = "Bob";
  this.age = 11;
  return {
    name: "LaLa"
  };
}
const foo = new Foo();
console.log(foo); // { name: 'LaLa' }

你可以对已定义的对象添加新的属性。例如，foo.grade = 'A' 语句会给 foo 添加一个新的属性 grade，并赋值为 A，而且不会影响到其他对象。

如果想要将新属性添加到相同类型的所有对象，你必须将该属性添加到 Foo 类型的定义中——

function Foo() {
  this.name = "Bob";
  this.age = 11;
}

const foo = new Foo();
const foo1 = new Foo();
console.log(foo.grade, foo1.grade); // undefined undefined

foo.grade = "A";
console.log(foo.grade, foo1.grade); // A undefined

Foo.prototype.grade = "B";
console.log(foo.grade, foo1.grade); // A B
